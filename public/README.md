# Public Assets Folder

## Logo Placement

Please place your official SATYAH logo file here with the name:
- `logo.png`

The logo should be:
- PNG format with transparent background
- High resolution (recommended: 512x512px or larger)
- Original proportions maintained

The logo will be automatically used in:
- Navigation bar (top-left)
- Hero section (centered)
- Footer
- Mobile menu
